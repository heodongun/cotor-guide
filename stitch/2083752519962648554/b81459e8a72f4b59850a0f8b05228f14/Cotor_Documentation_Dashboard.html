Cotor - AI-Driven Master Agent System

edit Edit this page

Cotor is a highly scalable, AI-driven master agent system designed to orchestrate complex workflows across multiple worker nodes. It leverages advanced LLMs to intelligently route tasks and manage state.

Quick Start link

Get up and running with Cotor in minutes. Ensure you have Python 3.9+ installed.

1. Installation

Install Cotor via pip:

bash 
pip install cotor-agent

2. Basic Configuration

Create a cotor.json file in your project root to define your initial master agent configuration.

json 
{
  "master": {
    "llm_provider": "openai",
    "model": "gpt-4-turbo",
    "max_concurrent_tasks": 5
  },
  "workers": [
    {
      "id": "worker_1",
      "capabilities": ["data_processing", "api_calls"]
    }
  ]
}

3. Initialize System

Use the Python SDK to start the master agent and load the configuration.

python 
from cotor import MasterAgent
# Initialize with the config file
agent = MasterAgent(config_path="cotor.json")
# Start the system
agent.start()
# Submit a task
result = agent.submit_task("Analyze user data from endpoint /api/users")
print(result)
